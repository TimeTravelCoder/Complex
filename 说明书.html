连接方式

连接方式 操作步骤 注意事项
数据线 长按home键可以关机 。电脑有线连接下有一个模式(Xbox、Dinput、Switch)，同时按住减号和加号2秒可以切换模式 。 使用数据线连接电脑的时候，充电很慢 。
接收器 首先将接收器插上设备的usb口 ，然后手柄在关机状态下，长按home键2秒钟不松手，直到手柄通道指示灯4灯快闪 。接收器会自动连上手柄(连上之后常亮) 。接收器连接下有2个模式(xbox模式和dinput模式)，同时按住减号和加号2秒可以切换模式 。 接收器比蓝牙更稳定，但是无法适配陀螺仪 。
蓝牙 xbox 模式 手柄在关机状态下，先按住X键不松手，然后再按住home键2秒也不松手，直到通道指示灯1灯快闪 。然后蓝牙搜索 "Xbox Wireless Controller" 。 xbox模式适配的游戏最多，优先使用这个模式 。
蓝牙 Switch 模式 手柄在关机状态下，先按住Y键不松手，然后再按住home键2秒也不松手，直到通道指示灯流水灯闪烁 。然后蓝牙搜索 "Pro Controller" 。 Switch模式适配的游戏少，不用陀螺仪可以不用这个模式 。
蓝牙 hid 模式 手柄在关机状态下，先按住(某个)键不松手，然后再按住home键2秒也不松手，直到通道指示灯3灯快闪 。然后蓝牙搜索 "GAMPAD" 。 这个模式是调节手柄的时候使用的，具体教程可以询问客服 。
蓝牙 V3映射 模式 手柄在关机状态下，先按住A键不松手，然后再按住home键2秒也不松手，直到通道指示灯2灯快闪 。然后蓝牙搜索 "GAMPAD V3" 。 支持安卓手机或者平板映射，具体教程可以询问客服 。
连接 Switch 游戏机 首先Switch上打开：手柄——更改握法/顺序 。然后手柄在关机状态下，先按住Y键不松手，然后再按住home键2秒也不松手，直到流水灯闪烁 。然后按照Switch上的提示进行连接，L和R(也就是左右肩键)同时按下 。可以按home键唤醒Switch 。 如果要投屏，那就先连上之后再投屏 。

高级功能设置

宏背键设置

  1. 进入编程模式: 按住(某个)键不放，设置的背键指示灯闪烁，表示手柄已经进入编程模式 。
  2. 设置连招: 按下下列任一想要被编程的按键，就是你的连招（加减键、T键、截图键、home键除外） 。
  3. 退出编程模式: 按下之前在第一步中按下的背键，指示灯恢复常亮，表示该背键编程成功 。
  4. 重置所有宏背键: 同时按住T键和加号键4秒不松手 。
注：ML默认键值为L3，MR默认键值为R3 。两个背键旁边的开关是背键锁，可以开关背键功能 。

连发设置

  1. 调节按键的状态: 先按住T键不松手，然后按一下想要设置连发的按键 。每个按键有三种状态：正常状态，按下连发状态，自动连发状态 。
  2. 连发速度调节: 按住手柄T键，同时进行右摇杆向上(增快)向下(减慢)，共三个档次，分别5/12/25次每秒 。
  3. 清除所有连发: 同时按住T键和减号键4秒不松手 。

灯光与震动调节

  • 灯光调节: 按下截图键可以切换灯光效果【炫彩-呼吸循环(红橙黄绿青蓝紫循环)-单色常亮-关闭】 。
    注：Switch模式下不支持切换灯光效果 。
  • 震动调节: 按住T键不松手，同时进行左摇杆向上(增强)向下(减弱)，共三个档次，100%震动、50%震动、关闭震动 。

电源与规格

  • 充电: 使用typec口数据线进行充电 。通道指示灯常亮表示当前剩余的电量，1格电20%，2格电50%，3格电80%，4格电100% 。
  • 关机: 当手柄在开机状态下，长按“home”键4秒进行关机 。
  • 复位功能: 当手柄出现异常时，可用牙签戳一下手柄背部的复位孔进行关机复位 。
  • 接收距离: 手柄有效接收距离8M以内 。
  • 电性规格:
    • 供电方式: 内置聚合物电池
    • 电池容量: 1000mAh
    • 使用时长: 6到10小时
    • 充电电压: DC5V
    • 最大充电电流: 484MA
  • 工作电流:
    • 休眠电流: 0uA
    • 无背景灯时: ≤25MA
    • 有背景灯时: ≤62MA
    • 有灯且双马达震动时: ≤135MA
